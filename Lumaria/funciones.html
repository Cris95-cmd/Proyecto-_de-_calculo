Curso de Funciones

1. Introducción a funciones

1.1 Concepto y notación

Definición:
Una función \(f\) es una regla de correspondencia, es decir, asocia a cada objeto \(x\) en un conjunto un único valor \(f(x)\) de un segundo conjunto.

Notación funcional

Una sola letra como \(f\) (ó \(g\) ó \(F)\) se utiliza para nombrar una función. Entonces \(f(x)\), que se lee \(f\) de \(x\) o \(f\) en \(x\), denota el valor que \(f\) asigna a \(x\). Por lo tanto, si \(f(x) = x^3 - 4\), entonces

\[f(2) = 2^{3} - 4 = 8 - 4 = 4\] \[f(a) = a^{3} - 4\] \[f(a+h) = (a+h)^{3} - 4 = \underbrace{a^{3} + 3a^{2}h + 3ah^{2} + h^{3}}_{\text{Desarrollo de } (a+h)^3} - 4\]

1.2 Graficación

Podemos describir la función mediante el trazo de su gráfica en un plano coordenado. La gráfica de una función \(f\) simplemente es la gráfica de la ecuación \(y\) = \(f(x)\).

La herramienta fundamental que usaremos en esta sección sera en plano cartesiano o tambien llamado plano coordenado.

Ejemplo para graficar la funcion \(f(x)\) = \(x^{2}\)

Como se dijo anteriormente \(y\) = \(f(x)\), quiere decir que si tomamos \(x\) = \(0\) obtendremos que \(y\) = \(f(0)\) y como se hizo en la seccion 1.1 \[y = f(0) = 0^{2} = 0 \]
Esto quiere decir que tomando \(x\) = \(0\) obtenemos que \(y\) = \(0\). Normalmente nos apoyamos de una tabla donde registraremos los puntos del plano cartesiano (x,y) y lo graficamos.

\(x\) \(y\)
\(0\) \(0\)
\(1\) \(1\)
\(2\) \(4\)
\(3\) \(9\)

1.3 Dominios y Rangos

Definiciónes:

  • El dominio D de una función es el conjunto formado por todos los valores que la función puede aceptar para devolver un único valor por cada uno de ellos.
  • El rango de una función es el conjunto formado por todos los valores que la función puede tomar.

    • Raices Cuadradas

    Cuando nuestra funcion sea una raíz cuadrada del tipo \(f(x) = \sqrt{x}\) debemos tener en cuenta que por como funciona una raíz cuadrada se puede definir como un numero que al multiplicarse por si mismo, resulta el numero original. O en otras palabras, si \(a\) es la raiz cuadrada de \(b\) (\(\sqrt{b} = a\)) es por que \(a \cdot a = b\).

    Como podemos ver, \(b\) siempre tendra un signo positivo, ya que por ley de signos si su raíz es negativa \(-a\) entonces \((-a) \cdot (-a)\) tendra un resultado positivo. Se puede decir que \((-a) \cdot (-a) = a \cdot a = b\).

    Dicho esto podemos analizar que en la funcion \(f(x) = \sqrt{x}\), va a ser imposible que x sea negativa, lo que quiere decir que mi funcion solo puede recibir valores positivos, o dicho de otra manera, el dominio de \(f(x)\) son todos los reales positivos \(Dom(f) = \Re^{+}\) o escrito de otra forma \(Dom(f) = [0,\infty) \).

    Por otro lado, una vez conocido el dominio, podremos saber su rango, ya que si nuestro dominio de \(f(x) = \sqrt{x}\) solo es positivo entonces su rango tambien sera positivo, ya que para todo valor positivo del dominio le correspondera un valor tambien positivo en el rango \(Ran(f) = \Re^{+}\) o escrito de otra forma \(Ran(f) = [0,\infty) \).

    Ejemplo

    \(f(x) = \sqrt{x-1}\)
    Ya que la raiz no puede operar un valor negativo podemos ver que \(x-1\) es negativo en todos los valores menores que \(x=1\), entonces el dominio \(Dom(f)=[1,\infty)\).
    Para su rango ya que cuando \(x=1\) , \(f(1) = \sqrt{1-1} = \sqrt{0} = 0\), a partir de \(x=1\) los valores de \(f(x)\) seran negativos, lo que quiere decir que su rango sera \(Ran(f) = \Re^{+}\) o escrito de otra forma \(Ran(f) = [0,\infty) \).

    Fracciones

    Cuando se nos presenta una funcion del tipo \(f(x) = \frac{1}{x}\), hay que dejar en claro que la división \(\frac{1}{0}\) es una indeterminación en las matematicas, ya que la división se define como \(\frac{a}{b} = c\) tal que \(c \cdot b = a\), esto quiere decir que para \(b=0\) entonces \(c \cdot 0 = a\) lo cual es estrictamente \(0\), pero sabemos que \(a \neq 0\) lo que quiere decir que no cumple con el criterio de la división.

    Esto queire decir que el dominio de la funcion \(f(x) = \frac{1}{x}\) seran todos los valores de \(x\) excepto cuando \(x=0\), entonces el dominio de \(f(x)\) son todos los reales menos el \(0\), \(Dom(f) = \Re \setminus \{0\} \), o escrito de otra forma \(Dom(f) = (-\infty,0) \cup (0,\infty)\)

    Para el rango de \(f(x)\) resultaria mas facil usando la funcion inversa, pero esta se enseñara hasta la seccion 3.7 del presente capitulo . Por ahora diremos que \(Ran(f) = \Re \setminus \{0\} \), o escrito de otra forma \(Ran(f) = (-\infty,0) \cup (0,\infty)\), el rango excluye al \(0\) debido a que la unica forma de que una fracción sea igual a 0 es solo si el numerador es \(0\) lo cual no se cumple en \(f(x) = \frac{1}{x}\).

    Ejemplo

    \(f(x) = \frac{1}{x+1}\)
    para saber el dominio de esta función debemos ver cuando el denominador sera \(0\), es decir, cuando \(x+1 = 0\), usando algebra tendremos que cuando \(x=-1\) sucede que \(x+1 = 0\), hecho esto podemos decir que el \(Dom(f) = \Re \setminus \{-1\}\) o \(Dom(f) = (-\infty,-1) \cup (-1,\infty)\).
    Para el rango debido a que la unica forma de que una fracción sea igual a 0 es solo si el numerador es \(0\) lo cual no se cumple en \(f(x) = \frac{1}{x+1}\) entonces el rango es \(Ran(f) = \Re \setminus \{0\} \), o escrito de otra forma \(Ran(f) = (-\infty,0) \cup (0,\infty)\).

    2. Tipos de funciones

    2.1 Algebraicas

    Las funciones algebraicas son aquellas funciones que se expresan mediante operaciones algebraicas básicas (suma, resta, multiplicación, división y potencias enteras) aplicadas a la variable independiente. Matemáticamente, una función f(x) es algebraica si puede describirse mediante una ecuación polinómica o una combinación de expresiones algebraicas.

    Estas funciones pueden tener diversas formas, entre las que se incluyen las funciones polinómicas (como \(f(x) = 3x^2 + 2x - 5)\), las funciones racionales (como \(f(x) = \frac{2x + 1}{x - 3}\)), y las funciones irracionales (como \(f(x) = \sqrt{x + 2}\)).

    2.2 Trascendentes

    Las funciones trascendentes son funciones que no satisfacen una ecuación polinómica, pero sus coeficientes son a su vez polinomios donde la variable independiente se ubica como exponente o como índice de la raíz, también se puede presentar afectada del signo logaritmo o de cualquiera de los signos que emplea la trigonometría que se vera en la siguiente sección.

    Estas funciones no están formadas por variables y números relacionados por operaciones algebraicas como suma, resta, multiplicación, división, potenciación y radicación.
    Esos son unos ejemplos: \[f(x)=e^{x}\] \[f(x)=5^{x}\] \[f(x)=log_{4}x\]

    2.3 Trigonométricas

    Definición:
    Supongamos que \(P=(x,y)\) es un punto del círculo unitario centrado en el origen \(O\). Supongamos que \(\theta\) es un ángulo con un lado inicial a lo largo del eje positivo \(x\) y un lado terminal dado por el segmento de línea \(OP\). Las funciones trigonométricas se definen entonces como

    • \(\large{sen \theta = y}\)
    • \(\large{cos \theta = x}\)
    • \(\large{tan \theta = \frac{y}{x}}\)
    • \(\large{cot \theta = \frac{x}{y}}\)
    • \(\large{sec \theta = \frac{1}{x}}\)
    • \(\large{csc \theta = \frac{1}{y}}\)
    Si los valores de \(x=0\), \(sec \theta\) y \(tan \theta\) son indefinidos. Si los valores de \(y=0\), entonces \(cot \theta\) y \(csc \theta\) son indefinidos.

    Para entender las funciones trigonometrícas nos apoyaremos de un triangulo rectangulo, mas precisamente, en uno que esta formado a travez de un circulo, con su primer vertice en el origen \(O\), el segundo se situara en el perimetro del circulo, y el tercero estara en eje \(x\) de tal forma que tenga un angulo de \(90°\), el angulo \(\theta\) se formara por los lados del triangulo que llamaremos \(Cateto\hspace{0.2cm}adyacente\hspace{0.2cm}e\hspace{0.2cm}Hipotenusa\), el lado restante se llamara \(Cateto\hspace{0.2cm}opuesto\) de esta forma decimos que:

    • \(\large{sen \theta = \frac{Cateto\hspace{0.2cm}opuesto}{Hiputenusa}}\)
    • \(\large{cos \theta = \frac{Cateto\hspace{0.2cm}adyacente}{Hiputenusa}}\)
    • \(\large{tan \theta = \frac{Cateto\hspace{0.2cm}opuesto}{Cateto\hspace{0.2cm}adyacente}}\)
    • \(\large{cot \theta = \frac{Cateto\hspace{0.2cm}adyacente}{Cateto\hspace{0.2cm}opuesto}}\)
    • \(\large{sec \theta = \frac{Hiputenusa}{Cateto\hspace{0.2cm}opuesto}}\)
    • \(\large{csc \theta = \frac{Hiputenusa}{Cateto\hspace{0.2cm}adyacente}}\)

    2.4 Especiales

    En las denominadas funciones especiales se encuentran funciones como la funcion a trozos o valor absoluto, que son funciones que vale la pena analizar.

    Funcion a trozos

    Una función a trozos, también llamada función a tramos, función segmentada o función seccionada, es aquella que se define con una expresión analítica diferente para distintos intervalos de su dominio. Se presentan de tal forma que se dividen en ramas, cada rama definida por una regla de correspondencia. \[f(x)= \left\{ \begin{array}{lcc} x^2 & si & x < 0 \\ \\ x-1 & si & x > 0 \end{array} \right.\]

    Funcion valor absoluto

    Una función de valor absoluto es una función que contiene una expresión algebraica dentro de los símbolos de valor absoluto. Recuerde que el valor absoluto de un número es su distancia desde 0 en la recta numérica . La función padre de valor absoluto, escrita como \(f(x) = |x|\), que se puede expresar como: \[f(x)= \left\{ \begin{array}{lcc} -x & si & x < 0 \\ \\ x & si & x \geq 0 \end{array} \right.\]

    3. Propiedades

    3.1 Raices

    Las raíces de una función \(y = f(x)\) son los valores \(x\) en los cuales \(f(x)\) se hace \(0\). En algunos casos, la función \(f\) tiene una forma tal que el problema se puede resolver algebraicamente. Es el caso de los polinomios de grado 2, e incluso los polinomios de grado 3 y 4. Sin embargo, en la mayoría de los casos, el problema sólo tiene solución numérica. Esto significa que hay que usar el computador para encontrar los valores numéricos de las raíces. No es posible, encontrar una fórmula algebraica que entregue las soluciones, en esta seccion solo se presentaran casos en donde la raiz tiene una solucion algebraica.

    Calculo de la raíz de una función

    Como se dijo, el la raíz de una funcion son los valores en donde \(f(x) = 0\) o en lo que se presentara en la grafica es el punto en donde la grafica de una funcion toca el eje de las \(x\), es decir, \(y = 0\).

    Tomemos como ejemplo la funcion \(f(x) = x\sup{3} - x\), para encontrar las raices de esta funcion debemos igualar la funcion a \(0\) y encontar los valores para los que \(x\) va a ser solución \begin{align*} 0 &= x^3 - x && \text{(Igualando la funcion a 0)} \\ 0 &= x(x^2 - 1) && \text{(Factorizando x)} \\ 0 &= x(x + 1)(x - 1) && \text{(Separando en diferencia de cuadrados)} \\ & \therefore \text{Las raíces de f(x) son en x=0,-1,1} \end{align*}

    3.2 Inyectividad, Suprayectividad y Biyectividad

    Inyectividad

    Definición: Una función \(f(x)\) es inyectiva (uno a uno) si cada elemento del rango (conjunto B) proviene de un único elemento del dominio (conjunto A), es otras palabras; no hay dos elementos del dominio que tengan la misma imagen.

    Proceso para verificar inyectividad
    La condición algebraica implica demostrar que si \(f(x) = f(y)\) , entonces \(x = y.\)

    Consideremos las siguientes funciones como ejemplo:

  • \(f(x) = x^2\)
    • Entendamos primero que \(f (x) = x^2\) y que \(f (y) = y^2\), la misma función solo que la variable se cambia por \(y\). Como paso \(1\) supongamos que es verdad que son iguales y veremos a que resultado llegamos.

    \[f(x) = f(y)\] \[x^2 = y^2\] al aplicar la raíz en ambos lados para simplificar nos quedará: \(\pm x = \pm y\) y cuatro combinaciones posibles: \[+x = +y \quad \checkmark\] \[-x = -y \quad \checkmark\] \[+x = -y \quad \times\] \[-x = +y \quad \times\] por lo tanto la función no es inyectiva.

  • \(f(x) = 2x + 1\)
    • Procedemos exactamente igual, recuerda: \(f(y) = 2y + 1\) y suponemos \[f(x) = f(y)\] \[2x + 1 = 2y + 1\] \[2x + \cancel{1} = 2y +\cancel{1} \quad \text{simplificamos}\] \[\cancel{2}x = \cancel{2}y\] \[x = y\] por lo tanto verificamos que si es biyectiva.

    Sobreyectividad

    Definición: Se dice que una función es suprayectiva, sobreyectiva o exhaustiva cuando el dominio (conjunto A) y rango (conjunto B) coinciden, no sobran elementos. Más formalmente todo elemento \(y\) (imagen) tiene al menos un elemento \(x\) (dominio) que le corresponde.

    Proceso para verificar suprayectividad:
    La condición implica demostrar que \(dom (f) = ran (f)\). Recordemos que en la sección 1.3 ya repasamos las bases para dominios y rangos de funciones.
    Observemos unos ejemplos:

  • \(f(x) = x^2\)

    • En la sección de Dominio y Rango de funciones dijimos que el \(Dom (f) = \Re\) y el \(Ran (f) = [0,\infty ^+)\), claramente no son iguales por lo que la función no es suprayectiva.

  • \(f(x) = x\)
    • Esta función mejor conocida como "función identidad" es por definición suprayectiva porque su \(Dom (f) = \Re\) y \(Ran (f) = \Re\), son iguales.

    Otra forma alternativa de comprobar suprayectividad dice:
    Una función \(f : X \rightarrow Y\) es suprayectiva si y solo si \(f\) tiene inversa derecha. Pero aun no llegamos a estudiar la sección de contención e inversa de funciones. Más adelante lo comprobaremos utilizando esta herramienta.

    Biyectividad

    Definición: Una función biyectiva es simplemente aquella que es inyectiva y suprayectiva a la vez, es decir, es de uno a uno y tiene el mismo dominio y rango (no sobran elementos).

    Para verificar biyectividad bastara demostar inyectividad y suprayectividad en la función.

    3.3 Par o Impar

    Función par

    Definición: Una función se dice que es par cuando tiene simétria con respecto al eje \(y\), en palabras más coloquiales si pudieramos doblar el plano cartesiano como una hoja de papel siendo el eje \(y\) la linea marcada, la función empalmaria exactamente

    Debera cumplir la condición: \(f(x) = f(-x)\) para cualquier \(x\) en el dominio. El nombre proviene de las potencias par de \(x\) es decir \(x^2,x^4,x^6...\) que son funciones pares. Supongamos por ejemplo que queremos corroborar que \(f(x) = x^4\) es par. \[¿f(x) = f(-x)?\]
    Comenzaremos evaluando: \begin{align*} f(-x) &= (-x)^4 && \text{(Aplicando la definición)} \\ &= x^4 && \text{(Simplificando)} \\ &= f(x) && \text{(Por definición de f)}\\ & \therefore \text{es par} \end{align*} Nota: Que una función tenga un exponente par no garantiza que sea par.

    Función impar

    Definición: Una función impar es aquella que tiene simétria central tambien llamada aveces "simétrica con respecto al origen \((0,0)\)". ¡Se ve igual si lo pones boca abajo! (... o si lo miras desde dos direcciones opuestas)

    Se llaman impares porque las funciones con exponentes impares (\(x, x^3, x^5, x^7,...)\) se comportan asi. Bastara demostrar que \(f(-x) = - f (x)\).
    Por ejemplo queremos corroborar que \(f(x) = x ^3\) es impar: \begin{align*} f(-x) &= (-x)^3 && \text{(Aplicando la definición)} \\ &= -x^3 && \text{(Simplificando)} \\ &= - f(x) && \text{(Por definición de f)}\\ \end{align*} verificamos que si es impar.

    3.4 Monotonía

    Máximos y mínimos

    Teorema de existencia de máximo y mínimo: Si \(f(x)\) es continua en un intervalo \([a,b]\), entonces \(f\) alcanza un valor máximo y un valor mínimo en ese intervalo.

    Seguro habras notado en algunas gráficas intervalos donde la función comienza a crecer, alcanza un punto máximo y decrece o por el contrario intervalos donde la función comienza a decrecer, alcanza un punto mínimo y crece nuevamente. A estos puntos \(c\) los llamaremos puntos criticos.

    3.5 Composición de funciones

    Definición: La composición de una función es una operación que combina dos o más funciones para crear una nueva función. Dadas dos funciones, \(f\) y \(g\), la composición se denota como \(f \circ g\) ó \(f(g(x))\), (\(g\) contenida en \(f\) leida de derecha a izquierda) y se define como: \[(f \circ g)(x) = f(g(x))\] Esto significa que primero se aplica la función \(g\) a la entrada \(x\), y luego se aplica la función \(f\) al resultado de \(g(x)\). Y por complemento se aplica el mismo método para \[(g \circ f)(x) = g(f(x))\]

    Sabemos que es un poco complejo comprenderlo, asi que lo ilustraremos un ejemplos:

  • Ejemplo 1: \(f (x) = 2x \quad\) y \(\quad g (x) = x^2\)
    • 1.\(f \circ g = f (g(x))\): significa que en vez de evaluar un escalar en la función evaluaremos una función completa. \begin{align*} f (x^2) &= 2(x^2)\\ &=2x^2\\ &\text{nuestra nueva función 1 seria \(2x^2\)}. \end{align*}
    2.\((g \circ f)(x) = g(f(x))\) \begin{align*} g (2x) &= (2x)^2\\ &=4x^2\\ &\text{nuestra nueva función 2 seria \(4x^2\)}. \end{align*}
  • Ejemplo 2: \(f (x) = 3x + 2\quad\) y \(\quad g (x) = \frac{x+3}{2x+1}\)
    • 1.\(f \circ g = f (g(x))\): \begin{align*} f (\frac{x+3}{2x+1}) &= 3(\frac{x+3}{2x+1}) + 2\\ &= \frac{3x+9}{2x+1} + 2\\ &= \frac{3x+9+2(2x+1)}{2x+1}\\ &= \frac{3x+9+4x+2}{2x+1}\\ &= \frac{7x+11}{2x+1}\\ &\text{nuestra nueva función 1 seria \(\frac{7x+11}{2x+1}\)}. \end{align*} 2.\(g \circ f = g (f(x))\): \begin{align*} g (3x) &= \frac{(3x)+3}{2(3x)+1}\\ &= \frac{3x+3}{6x+1}\\ &\text{nuestra nueva función 2 seria \(\frac{3x+3}{6x+1}\)}. \end{align*}

    3.6 Inversa de funciones

    Definición: Una función \(f\) es invertible si y solo si existe una función \(g\) tal que \(f(g(x)) = x\) para todo \(x\) en el dominio de \(g\), y \(g(f(x)) = x\) para todo \(x\) en el dominio de \(f\).

    Vamos por partes, la inversa de una función \(f(x)\) la podemos denotar como \(f^{-1}(x)\). Solamente si y solo si la función es biyectiva podremos decir que admite o "tiene" inversa y funcionarán de la siguiente manera:
  • \(f : Dom (f) \rightarrow Img (f)\)
    • La función tomará un elemento del dominio y regresará uno de la imagen.
  • \(f^{-1}: Img (f) \rightarrow Dom (f)\)
    • Su inversa tomara el elemento de la imagen que \(f(x)\) regreso y nos dará de nuevo el elemento del dominio. Tal como suena es un circulo de entradas y salidas

  • ¿Porque es importante que \(f(x)\) sea biyectiva?

    • Biyectiva = Inyectiva + Sobreyectiva \(\rightarrow\) Garantiza que \(f^{-1}(x)\) sea una función (no multivaluada) y esté definida en toda la imagen. Si \(f(x)\) no es biyectiva, su "inversa" tendría problemas:

    1. Puede no ser una función dos elementos de la imagen llegarian a una del dominio y eso no es una función (si no es inyectiva).
    2. Puede no estar definida en toda la imagen, para \(f(x)\) un \(x \rightarrow y\quad\) pero para \(f^{-1}(y)\) un\(\quad y \rightarrow ?\) podria no tener elemento de regreso (si no es sobreyectiva).

    Bastaria con demostrar que la función es biyectiva y sabremos que admite inversa y/ó encontrar la inversa y demostrar que tiene inverso derecho e izquierdo por contención como lo vimos en la sección anterior.
    \[f(f^{-1}(y)) = y\] \[f^{-1}(f(x))= x \] haciendo un poco de algébra.

  • Ejemplo 1: Proceso para encontar inversa
    • Comencemos prácticando un poco e intentemos demostrar que \(f(x) = x^2\) no admite inversa (recuerda que ya habiamos demostrado que es una función no biyectiva).
    Paso 1.Despejamos para \(x\). \begin{align*} x^2 &= y\\ x &= \sqrt{y}\\ f^{-1}(y) &= \sqrt{y} \end{align*}
    Paso 2. Verificar inverso derecho por contención de funciones. \begin{align*} f(f^{-1}(y)) &= f (\sqrt{y})\\ &= (\sqrt{y})^2\\ &= y \quad\checkmark \end{align*}

    Paso 3. Verificar inverso izquierdo. \begin{align*} f^{-1}(f(x)) &= f^{-1}(x^2)\\ &= \sqrt(x^2)\\ &= \left|x\right|\quad \times \end{align*} notese que el valor absoluto de \(x\) no es lo mismo que \(x\). Por lo tanto no tiene inverso izquierdo lo que implica que no tiene inversa general.

    4. Transformación de funciones

    4.1 Traslación de funciones

    Definición:Trasladar una función dentro del plano cartesiano significa desplazarla vertical u horizontalmente sin alterar la forma u orientación de la función, aunque si cambiaremos su dominio y rango.

    Comencencemos entendiendo los traslados verticales. Podemos desplazar nuestra función hacia arriba o abajo respecto a el eje \(y\)

  • \(\uparrow\) para subirla sumamos un escalar \(k\) a la función: \(f(x) + k\)
  • \(\downarrow\) para bajarla restamos un escalar \(k\) a la función: \(f(x) - k\)
    •

    Los traslados horizontales en cambio funcionarán un poco diferente, tendremos que sumar o restar el valor \(k\) directo a la variable \(x\), es decir:

  • \(\rightarrow\) moverla a la derecha restamos el valor \(k\) : \(f(x-k)\)
  • \(\leftarrow\) moverla a la izquierda sumamos el valor \(k\) : \(f(x+k)\)
    •

    4.2 Reflexiones respecto a ejes

    Definición: La reflexión de una función respecto a un eje implica crear una imagen espejo de la gráfica original, utilizando el eje como referencia. La reflexión respecto al eje \(x\) se obtiene cambiando el signo de la función \(y = - f(x)\).



  • Ejemplo:
    • Si tienes la función \(y = x²\), la reflexión horizontal sería \(y = (-x)²\), que es lo mismo que \(y = x\), porque el cuadrado elimina el signo negativo.

    Definición: La reflexión respecto al eje \(y\) se obtiene cambiando el signo de la variable independiente \(y=f(-x)\).

    4.3 Estiramientos y compresiones

    Estiramiendo o compresión vertical

    Definición: Dada una función \(f(x)\), una nueva función \(g(x) = a f(x)\), donde \(a\) es una constante, es un estiramiento vertical o compresión vertical de la función \(f(x)\). Donde la constante \(a\) hace más ancha o más delgada la función.

  • Si los valores de \(a > 1\), entonces el gráfico se estirará.


  • Si los valores de \(0 < a < 1\) , entonces el gráfico se comprimirá.


  • Si los valores de \(a < 0 \) , entonces habrá la combinación de estiramiento o compresión vertical con reflexión vertical.

    Estiramiendo o compresión horizontal

    Definición: Dada una función \(f(x)\), una nueva función \(g(x)=f(b x)\),donde \(b\) es una constante, es un estiramiento horizontal o una compresión horizontal de la función \(f(x)\).

  • Si los valores de \(b > 1\), entonces el gráfico se comprimirá por \(\frac{1}{b}\).
  • Si los valores de \(0 < b < 1 \), entonces el gráfico se estirará por \(\frac{1}{b}\).
  • Si \( b < 0 \), entonces habrá una combinación de estiramiento o compresión horizontal con reflexión horizontal.
